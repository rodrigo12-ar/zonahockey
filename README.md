# Zona Hockey Store

Proyecto base de e-commerce para una tienda de hockey con Next.js, Tailwind CSS y TypeScript.

## Ejecutar localmente

1. Instalar dependencias

```bash
npm install
```

2. Iniciar servidor de desarrollo

```bash
npm run dev
```

3. Abrir en el navegador

```bash
http://localhost:3000
```

## Características incluidas

- App Router de Next.js
- Tailwind CSS y diseño oscuro deportivo
- Productos físicos y fotografías
- Catálogo público con detalles y carrito
- Checkout inicial con selección de pago
- Panel de administración base para CRUD
- Estructura preparada para integrar Supabase/PostgreSQL y Mercado Pago
